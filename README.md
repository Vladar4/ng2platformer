Nimgame 2 Platformer
====================

A [tutorial](https://vladar4.github.io/nimgame2/tut102_platformer.html) game for the [Nimgame 2](https://github.com/Vladar4/nimgame2) engine.

Controls:
---------

| Action        | Control     |
| --------------|-------------|
| Move left     | Arrow Left  |
| Move right    | Arrow Right |
| Jump          | Space       |
| Exit to menu  | Escape      |
